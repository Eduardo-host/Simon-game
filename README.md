# Simon Game

A classic Simon memory game built with HTML, CSS, JavaScript, and jQuery.

## Preview

The game challenges the player to remember and repeat an increasingly long color sequence.  
It also includes a persistent **Best Score** feature using `localStorage`.

## Features

- Interactive Simon game logic
- Sound effects for each color
- Button animations
- Game over effect
- Persistent best score saved in the browser
- Clean and simple UI

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery

## How to Play

1. Press any key to start the game.
2. Watch the color sequence carefully.
3. Repeat the sequence by clicking the buttons in the correct order.
4. Each round adds one new color to the pattern.
5. If you click the wrong color, the game ends.
6. Try to beat your best score.
